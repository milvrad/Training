4. Starting the Weekly Schedule

Learn → guided workflow → tool practice → knowledge check → competency evidence

📋
1 · Learn

Review the core rules

Read the lesson cards in order. New terms are defined in the Terminology & Abbreviation Guide.

2 · Practice

Apply the workflow

Use the guided workflow and the required tool-practice activity before taking the quiz.

3 · Demonstrate

Show competency evidence

Competency is based on quiz performance plus documented use of the scheduling tools.

Learning objectives

  1. Gather the required weekly printouts and working tools.
  2. Export and use QGenda vacation/time-off information correctly.
  3. Prepare the Schedule Check Grid before placing flexible work.
  4. Place fixed homes, admins, IR locations, default sites, and known evening obligations first.

Required weekly tools

Start with the Schedule Check Grid, Rad Weekly Evening Schedule, Neuro Rads Fluoro Weekly Schedule, Weekly Sheet, and Weekly Seating Chart.

Vacation/time-off export

In QGenda, go to Admin, use Filters on the left, select Tasks → Vacation/Time Off, then use Reports to export for the Monday-Friday week. Turn off all tabs except pending requests, run the report, open it in Excel, print on one page, and X off the Schedule Check Grid.

Fixed placements

Before running or manually filling flexible work, place fixed items: IR rads and locations, home assignments, admin days, fixed locations, calendars, tumor boards, weekly seating needs, and always-evening rads.

Default locations

Known defaults include LVH-S as GE and AML, HAZ as LB, HWC as WH, and 17th Street as KS on Monday/Tuesday/Thursday/Friday, if working and not otherwise changed.

Evening markers

Mark evening obligations on the Schedule Check Grid: 1-10 in AM slot, 12-9 in AM slot, 4-12 in AM/PM slots, AM/EVE in PM slot with an asterisk, and PMC 12:30-9 in AM/PM slots.

Core training content

Before opening the schedule

Start by gathering the printed/working tools: Schedule Check Grid, Rad Weekly Evening Schedule, Neuro Rads Fluoro Weekly Schedule, Weekly Sheet, Weekly Seating Chart, vacation/time-off report, home/admin lists, and any current quarter grids.

Vacation/time-off export

  1. In QGenda, go to Admin.
  2. Use Filters → Tasks → Vacation/Time Off.
  3. Use Reports and export one week at a time, Monday through Friday.
  4. Turn off all tabs except pending requests if that is the local workflow.
  5. Print from Excel, scaled to one page, and mark the Schedule Check Grid.

Fixed placements

  • Place IR locations on the Schedule Check Grid and confirm IR is seated for the week.
  • Highlight home days and add Home in QGenda.
  • Add admin days to both the Schedule Check Grid and QGenda.
  • Use known fixed/default sites only when the current rule still applies.

Evening markers before day build

Mark rads already placed or always evening before running the rest of the weekly schedule. Examples include 1-10, 12-9, 4-12, AM/EVE, and PMC 12:30-9.

Guided workflow
  1. Print or open all weekly working tools.
  2. Export vacation/time off from QGenda for Monday-Friday.
  3. X off unavailable blocks on the Schedule Check Grid.
  4. Set QGenda to the week being worked.
  5. Mark all IR rads and confirm each is seated for the full week.
  6. Add homes and admins to the grid and QGenda.
  7. Add default fixed locations where applicable.
  8. Mark evening obligations before running schedule logic.
